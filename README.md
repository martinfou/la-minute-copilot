# 🎤 La minute Copilot

Votre session hebdomadaire de 5 minutes pour maîtriser Microsoft Copilot au quotidien.
À l'intention des employés IT et non-IT.

## Structure

```
la-minute-copilot/
├── presentations/      ← PowerPoints prêts à présenter (.pptx)
├── scripts/            ← Scripts Python pour générer les présentations
├── prompts/            ← Archive des prompts et cheat sheets
└── assets/             ← Images, logos, templates
```

## Épisodes

| # | Sujet | Date | Fichier |
|---|-------|------|---------|
| 1 | Outlook — 3 astuces + sauvegarder ses prompts | 2026-05-17 | `presentations/La_Minute_Copilot_Outlook.pptx` |

## Générer une présentation

```bash
cd scripts && python3 generate.py
```
